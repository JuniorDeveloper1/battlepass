# Battlepass Plugin
Author : JuniorDeveloper.

## Commands
* **/bp level :** 
 
 **Description :** Shows your level
  
* **/bp challanges :** 
 
   **Description:** Shows challanges!    
  
*  **bp check <player> :** 
  
 **Description :** Checks an player level!
*   **bp xp list :**
  
 **Description:** Shows every level up. With xp and Level! **For example :** Level 1 1000XP!
